Tox Prediction

An Intelligent Approach to Predict Toxicity in the field of Cheminformatics

This project is about analyzing and modeling ToxCast and Tox21 data to investigate food-related chemicals as it is crucial for public health. Detailed biological experiments have been carried out to determine whether the chemical is toxic or not, but this method adds up the cost. Therefore, we propose to automate the process of toxicity prediction using machine learning and deep learning models.
To achieve this we started with Data Preprocessing which includes null value treatment, data balancing and data visualization.
